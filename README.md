sunplot
=======

App for plotting sunrises and sunsets

Instructions to get going:

0. Node must be installed 
1. Open up your terminal and do 'git clone https://github.com/IanPirro/sunplot.git' 
2. The navigate to the sunplot directory 
3. In that folder do 'npm install' ( I included the bower stuff in the repo no need to bower install )
4. When npm install finishes successfully run 'grunt'
5. Navigate to http://localhost:3000 in the browser
